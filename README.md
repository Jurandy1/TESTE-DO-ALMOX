# Sistema de Controle de Almoxarifado - SEMCAS

Sistema de gestão de almoxarifado, controle de estoque (água, gás, materiais), requisições e relatórios.

## Funcionalidades
- Controle de Estoque (Entradas e Saídas)
- Gestão de Requisições de Materiais
- Leitor de Requisições (Importação de Excel/Word)
- Dashboards e Relatórios
- Controle de Acesso (Admin/Editor/Anônimo)

## Tecnologias
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (ES6 Modules)
- Firebase (Firestore, Auth, Storage)
